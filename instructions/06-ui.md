# UI & BRANDING

The application must follow the Solution Boxes brand identity consistently across every page, component, and interaction. Never use random colors, fonts, layouts, or styles.

The design should feel like a premium SaaS product similar to ChatGPT, Claude, or Perplexity, while maintaining the Solution Boxes identity.

**Brand name:** Solution Boxes SEO Agent
**Company:** Solution Boxes — https://solutionboxes.com

## Design Philosophy

The interface must be modern, minimal, professional, premium, clean, fast, trustworthy, AI powered, easy to use, mobile friendly, and responsive.

- Avoid clutter.
- Use whitespace generously.
- Every screen should feel clean and comfortable.

## Brand Colors

| Token | Hex | Use for |
|---|---|---|
| Primary (orange) | `#FF7A00` | Primary buttons, icons, active states, links, highlights, loading animations, focus states. Use as an accent — never overuse. |
| Secondary (navy) | `#071B2C` | Headings, navigation, logo, important text, sidebar |
| Background | `#FFFFFF` | Clean white app background |
| Light background | `#F8FAFC` | Cards and subtle sections |
| Border | `#E5E7EB` | Thin modern borders |
| Text primary | `#0F172A` | |
| Text secondary | `#475569` | |
| Text muted | `#94A3B8` | |

## Typography

- Use the Inter font.
- Modern typography with generous spacing.
- Large headings, readable paragraph spacing.
- Never use decorative fonts.

## Layout

- Centered, responsive layout. Max content width ~1400px.
- Rounded corners, soft shadows, modern cards.
- No unnecessary gradients, no flashy effects, no glassmorphism, no skeuomorphism.
- Keep everything simple.

## Header

- Displays "Solution Boxes SEO Agent".
- Include a small AI indicator.
- Display a green status badge: `● Live`.

## Logo Area

- Solution Boxes branding style: dark navy text, orange highlights.
- Minimal, professional appearance.

## Chat Interface

Feel similar to ChatGPT. Includes:

- Streaming responses
- Typing animation
- Markdown support
- Tables
- Code blocks
- Copy response button
- Regenerate button
- Auto scroll
- Responsive layout
- Conversation history

## Chat Input

- Rounded, modern input, large enough for comfortable typing.
- Orange send button.
- Keyboard shortcuts.
- Auto resize.

## Sidebar

Includes: New Chat, Recent Chats, Search Conversations, Settings, Documentation, About. Supports collapsing.

## Buttons

**Primary:** orange background, white text, rounded, medium shadow, hover animation.
**Secondary:** white background, orange border, orange text.

## Icons

Outline icons, orange accent, simple, professional.

## Cards

Rounded corners (16px radius), soft shadow, clean spacing, white background.

## Animations

Subtle only: fade, slide, streaming text, smooth transitions. Never use distracting animations.

## Mobile Experience

Fully responsive, ChatGPT-like mobile experience, sticky chat input, touch friendly, optimized spacing.

## Accessibility

High color contrast, keyboard navigation, proper focus states, readable font sizes, accessible buttons.

## Brand Experience

Every screen should communicate: professional SEO expertise, trust, reliability, modern AI technology, fast support, premium software quality.

The overall feeling should be similar to premium SaaS platforms such as ChatGPT, Claude, Notion, Linear, and Vercel, while remaining unique to the Solution Boxes brand.

Always follow this branding guide unless explicitly instructed otherwise.
