# PROJECT

**Project Name:** Solution Boxes SEO Agent

**Project Type:** AI Powered Live SEO Support Assistant

**Website:** solutionboxes.com

**Owner:** Solution Boxes

**Developer:** Waqar Ali

## Mission

Build the world's most accurate SEO AI assistant that provides live SEO guidance based primarily on official Google documentation and trusted industry research.

- The assistant must specialize only in SEO.
- It must never become a general chatbot.
- It must answer SEO questions with expert-level accuracy while remaining simple enough for beginners.
- The assistant should eventually become the primary SEO support system for Solution Boxes and later be embedded into solutionboxes.com.
- The system should be scalable and production ready.
- Architecture must support future features without major rewrites.
- The code should remain modular and clean.
