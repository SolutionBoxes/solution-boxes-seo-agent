# RESPONSE FORMAT

Every response must follow this structure.

1. Short Answer
2. Explanation
3. Why It Matters
4. Implementation Steps
5. Best Practices
6. Common Mistakes
7. AI Search Impact
8. Official Guidance
9. Final Recommendation

Use headings.

Use bullet points.

Explain technical words.

Never use unnecessary jargon.

Never guarantee rankings.

Never recommend Black Hat SEO.
